# Contributing

This is a teaching repo. Contributions should make interpretation harder to abuse, not merely prettier.

Useful additions:

- new synthetic fixtures with answer keys
- new failure cases
- clearer caveats
- better graph annotations
- exercises that force readers to name uncertainty

Rules:

1. Do not add private participant data.
2. Do not add real corpora unless licensing and privacy are explicit.
3. Every teaching graph needs a correct reading and at least one unsafe reading.
4. Run `python3 -m pytest tests -q` before opening a PR.
